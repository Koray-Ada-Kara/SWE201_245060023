# Proje

## Description
This project is a simple C# console application created for an assignment.
It demonstrates basic object-oriented programming concepts.

## Files
- Program.cs: Contains the Main function and program execution logic.
- proje.cs: Defines the Kisi class with properties, constructors, and methods.

## Development Steps
1. Created a class named Kisi.
2. Added two properties: Isim and Yas.
3. Implemented an empty constructor and a parameterized constructor.
4. Added a member function and overrode the ToString() method.
5. Created objects in the Main function, performed operations, and printed the results.
